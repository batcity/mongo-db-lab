# MongoDB Learning Lab (MongoDB-first Structure)

This repo is a **MongoDB-focused learning lab**.  

## Quickstart

    docker compose up -d
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt

Uses a **single-node replica set** so transactions & change streams work locally.

## Labs

- indexing
